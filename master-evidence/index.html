    ETC6 Master Evidence File¶

    This file tracks ALL unique evidence items across the etc-XX series (The Final Fate of the Wicked). Each item is registered once with a unique ID. Individual studies reference items by their master ID.

    How to Use This File¶

    1. Before writing your CONCLUSION.md tally, read this file to check for existing items
    2. For each E/N/I item in your tables:
    3. If an item with the same verse + same observation already exists → note the master ID
    4. If no match exists → add it here with the next available ID
    5. Update the "First Appeared" and "Also In" fields as appropriate
    6. Run Tree 3 (E-Positional) for every E-item before assigning a position
